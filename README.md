# Assistant

A new type of termianl based on [electron](https://github.com/atom/electron).  Original idea can be found [here](https://github.com/29decibel/assistant).  Inspired heavily from [hubot](https://github.com/github/hubot) and [Om.next](https://github.com/omcljs/om).

### Dependencies

```bash
npm install electron-prebuilt -g
```

### Development

```bash
npm install
npm run app
```
